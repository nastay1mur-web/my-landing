# Правила работы с проектом my-landing

## Стек и архитектура

- Всё в одном HTML-файле — стили в `<style>`, скрипты в `<script>`, никаких внешних JS-библиотек
- Шрифты можно подключать через Google Fonts (единственное исключение)
- Никаких фреймворков (React, Vue, Alpine и т.п.) — только чистый HTML + CSS + Vanilla JS

## Адаптивность

- Мобильная версия обязательна — проверяй вёрстку от 375px
- Используй `clamp()` для шрифтов и отступов, где это уместно
- Breakpoints: 375px / 768px / 1024px / 1440px
- Кнопки на мобильном — полная ширина (`width: 100%`)
- Кастомный курсор и canvas-частицы — только на desktop (`pointer: fine`)

## Стиль текста

- Простой, живой язык — как будто объясняешь другу
- Никакого канцелярита: не "осуществляет разработку", а "делает сайты"
- Короткие предложения, глаголы вместо отглагольных существительных
- Цифры и конкретика вместо размытых обещаний

## Дизайн-система

- Фон: `#050510` / `#0A0A1A` / `#12122A`
- Акценты: `#6366F1` (indigo), `#00D4FF` (cyan), `#EC4899` (pink)
- Текст: `#F8FAFC` / `#94A3B8` / `#475569`
- Шрифты: Space Grotesk (заголовки), Inter (текст), JetBrains Mono (теги/код)
- Карточки: glassmorphism (`backdrop-filter: blur`, `rgba` фон, тонкая рамка)
