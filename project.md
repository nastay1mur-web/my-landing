# project.md — Описание проекта

---

## Что это за проект

Лендинг-портфолио цифрового специалиста, который создаёт сайты, веб-приложения и AI-агентов под ключ. Сайт — главный инструмент привлечения клиентов: показывает услуги, кейсы и экспертизу, снимает типичные возражения и ведёт к одному действию — написать в Telegram или заполнить форму.

---

## Для кого

Четыре сегмента аудитории:

| Сегмент | Кто | Задача |
|---|---|---|
| Малый бизнес | Владелец кафе, студии, магазина | Нужен сайт чтобы клиенты находили |
| Средний бизнес | Операционный директор, e-commerce | Автоматизировать заявки и рутину |
| Стартап | Основатель, продакт | Надёжный разработчик для MVP |
| Корпоративный | Digital-директор | Аутсорс разработки |

Приоритет — малый и средний бизнес, которые хотят результат без погружения в технологии.

## Какие секции уже есть (всё в одном файле — index.html)

| Секция | Якорь | Что внутри |
|---|---|---|
| Шапка | `header` | Логотип, навигация, CTA-кнопка, hamburger на мобильном |
| Hero | `#hero` | Заголовок, подзаголовок, 2 кнопки, 4 цифры-статистики, canvas-частицы |

| Услуги | `#services` | 3 карточки: сайты, приложения, AI-агенты — с ценами и списком возможностей |
| Работы | `#work` | Фильтр по типу + 4 кейса с результатами |
| Процесс | `#process` | 5 шагов работы + блок гарантий |
| AI-демо | `#ai-demo` | Живой чат-виджет (mock), описание возможностей агента |
| Технологии | `#stack` | 4 группы стека + бегущая строка значков |
| Отзывы | `#testimonials` | 3 карточки с цитатами, именами, должностями |

## Какие страницы планируются

Из `PLAN.md`:

- **Визитка** — мини-страница для быстрого знакомства (отдельный файл)
- **Страница контактов** — расширенная, с картой и формой
- **Портфолио** — реальные кейсы с цифрами и деталями
- **Страница «Спасибо»** — после отправки формы

## Стиль дизайна

**Aurora Dark UI** — тёмный фон с indigo/cyan свечениями и glassmorphism-карточками.

| Элемент | Значение |
|---|---|
| Фон | `#050510` / `#0A0A1A` / `#12122A` |
| Акцент 1 | `#6366F1` — Indigo |
| Акцент 2 | `#00D4FF` — Cyan |
| Акцент 3 | `#EC4899` — Pink |
| Текст | `#F8FAFC` / `#94A3B8` / `#475569` |
| Заголовки | Space Grotesk |
| Текст | Inter |
| Теги / код | JetBrains Mono |
| Карточки | Glassmorphism (`backdrop-filter: blur`, полупрозрачный фон, тонкая рамка) |
| Анимации | CSS transitions + IntersectionObserver + Canvas (частицы) |

Ориентир по ощущению: дорого, современно, технологично — но без холодности. Живой текст, человеческий тон.

## Технический стек

Чистый HTML + CSS + Vanilla JS в одном файле. Без фреймворков и внешних библиотек (только Google Fonts).
